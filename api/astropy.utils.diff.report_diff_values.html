report_diff_values¶

astropy.utils.diff.report_diff_values(a, b, fileobj=<_io.TextIOWrapper name='<stdout>' mode='w' encoding='utf-8'>, indent_width=0)[source]¶

Write a diff report between two values to the specified file-like object.

Parameters
a, b

Values to compare. Anything that can be turned into strings and compared using difflib should work.

fileobjobject

File-like object to write to. The default is sys.stdout, which writes to terminal.

indent_widthint

Character column(s) to indent.

Returns
identicalbool

True if no diff, else False.
