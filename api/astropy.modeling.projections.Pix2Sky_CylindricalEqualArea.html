Pix2Sky_CylindricalEqualArea¶

class astropy.modeling.projections.Pix2Sky_CylindricalEqualArea(*args, **kwargs)[source]¶

Bases: astropy.modeling.projections.Pix2SkyProjection, astropy.modeling.projections.Cylindrical

Cylindrical equal area projection - pixel to sky.

Corresponds to the CEA projection in FITS WCS.

\[\begin{split}\phi &= x \\ \theta &= \sin^{-1}\left(\frac{\pi}{180^{\circ}}\lambda y\right)\end{split}\]
Parameters
lamfloat

Radius of the cylinder in spherical radii, λ. Default is 1.

param_names = ('lam',)¶

Names of the parameters that describe models of this type.

The parameters in this tuple are in the same order they should be passed in when initializing a model of a specific type. Some types of models, such as polynomial models, have a different number of parameters depending on some other property of the model, such as the degree.

When defining a custom model class the value of this attribute is automatically set by the Parameter attributes defined in the class body.
