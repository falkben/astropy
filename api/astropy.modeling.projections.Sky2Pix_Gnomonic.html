Sky2Pix_Gnomonic¶

class astropy.modeling.projections.Sky2Pix_Gnomonic(*args, **kwargs)[source]¶

Bases: astropy.modeling.projections.Sky2PixProjection, astropy.modeling.projections.Zenithal

Gnomonic Projection - sky to pixel.

Corresponds to the TAN projection in FITS WCS.

See Zenithal for a definition of the full transformation.

\[R_\theta = \frac{180^{\circ}}{\pi}\cot \theta\]
