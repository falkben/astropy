get_xml_encoding¶

astropy.utils.xml.iterparser.get_xml_encoding(source)[source]¶

Determine the encoding of an XML file by reading its header.

Parameters
sourcepath-like object, readable file-like object file-like object, or callable()

Handle that contains the data or function that reads it. If a function or callable object, it must directly read from a stream. Non-callable objects must define a read method.
