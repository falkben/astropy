FixedWidth¶

class astropy.io.ascii.FixedWidth(col_starts=None, col_ends=None, delimiter_pad=' ', bookend=True)[source]¶

Bases: astropy.io.ascii.Basic

Fixed width table with single header line defining column names and positions.

Examples:

# Bar delimiter in header and data

|  Col1 |   Col2      |  Col3 |
|  1.2  | hello there |     3 |
|  2.4  | many words  |     7 |

# Bar delimiter in header only

Col1 |   Col2      | Col3
1.2    hello there    3
2.4    many words     7

# No delimiter with column positions specified as input

Col1       Col2Col3
 1.2hello there   3
 2.4many words    7

See the Fixed-Width Gallery for specific usage examples.
