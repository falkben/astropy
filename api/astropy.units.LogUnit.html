LogUnit¶

class astropy.units.LogUnit(physical_unit=None, function_unit=None)[source]¶

Bases: astropy.units.function.FunctionUnitBase

Logarithmic unit containing a physical one

Usually, logarithmic units are instantiated via specific subclasses such MagUnit, DecibelUnit, and DexUnit.

Parameters
physical_unitUnit or string

Unit that is encapsulated within the logarithmic function unit. If not given, dimensionless.

function_unitUnit or string

By default, the same as the logarithmic unit set by the subclass.
