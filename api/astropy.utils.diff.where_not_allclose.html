where_not_allclose¶

astropy.utils.diff.where_not_allclose(a, b, rtol=1e-05, atol=1e-08)[source]¶

A version of numpy.allclose() that returns the indices where the two arrays differ, instead of just a boolean value.

Parameters
a, barray_like

Input arrays to compare.

rtol, atolfloat

Relative and absolute tolerances as accepted by numpy.allclose().

Returns
idxtuple of array

Indices where the two arrays differ.
