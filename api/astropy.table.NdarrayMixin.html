NdarrayMixin¶

class astropy.table.NdarrayMixin(obj, *args, **kwargs)[source]¶

Bases: numpy.ndarray

Mixin column class to allow storage of arbitrary numpy ndarrays within a Table. This is a subclass of numpy.ndarray and has the same initialization options as np.array().
