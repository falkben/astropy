IllegalMinuteError¶

exception astropy.coordinates.IllegalMinuteError(minute)[source]¶

Raised when an minute value is not in the range [0,60].

Parameters
minuteint, float

Examples

if not 0 <= min < 60:
    raise IllegalMinuteError(minute)
