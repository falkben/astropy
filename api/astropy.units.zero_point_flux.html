zero_point_flux¶

astropy.units.zero_point_flux(flux0)[source]¶

An equivalency for converting linear flux units (“maggys”) defined relative to a standard source into a standardized system.

Parameters
flux0Quantity

The flux of a magnitude-0 object in the “maggy” system.
