MergeNpConcatenate¶

class astropy.utils.metadata.MergeNpConcatenate[source]¶

Bases: astropy.utils.metadata.MergeStrategy

Merge left and right objects using np.concatenate. This merge strategy is globally enabled by default.

This will upcast a list or tuple to np.ndarray and the output is always ndarray.
